# ⚡ Miracle Chain Connect
A simple, one-click tool to add the **Miracle Chain Mainnet** to MetaMask and check your **$PNIC** balance instantly.

## 🚀 Live Demo
[Click here to use the tool](https://vincenthaakmat.github.io/miracle-chain-tool/)

## 🛠 Features
- **Auto-Switch:** Automatically switches to Miracle Chain if already added.
- **One-Click Add:** Adds Miracle Chain Mainnet (Chain ID: 1247) to MetaMask.
- **Balance Checker:** Real-time $PNIC balance display.

## 🔗 Network Details
- **Network Name:** Miracle Chain Mainnet
- **RPC:** https://miracle-rpc.calderachain.xyz/http
- **Chain ID:** 1247
- **Symbol:** PNIC
